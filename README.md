
🌱 YouThrive

A full stack habit tracking web application that helps users build consistent routines and understand their daily progress through data, visualizations, and personalized AI feedback.

Check it out: https://youthrive-habit-tracker.vercel.app

📋 Overview

YouThrive is a full stack habit tracking web application that helps users build consistent routines and better understand their daily progress. Along with creating and managing habits, users can view interactive analytics, track their progress over time, and receive personalized AI insights based on their activity. The goal was to build an application that goes beyond simply checking off habits by providing meaningful feedback in a clean, intuitive interface.

🛠️ Tech Stack


Frontend: React, Vite, Axios, Recharts
Backend: Node.js, Express.js
Database: MongoDB, Mongoose
Auth: JWT Authentication
AI: Google Gemini API


💡 The Problem

Many habit tracking apps focus only on recording completed habits. While that helps users stay organized, it doesn't provide much insight into their progress or encourage long-term consistency. I wanted to create an application that helps users understand their habits through data, visualizations, and personalized feedback.

✅ The Solution

YouThrive allows users to securely create an account, manage their habits, organize them with drag and drop, track daily progress, view interactive charts, and receive AI-generated insights based on their activity. Every feature was designed to make habit tracking simple while giving users a clearer picture of how they're improving over time.

✨ Features


🔐 Secure JWT authentication
📝 Create, edit, delete, and complete habits
🖱️ Drag and drop habit organization
📊 Interactive charts and progress analytics
🤖 AI-generated habit insights using Google Gemini
🔗 REST API built with Express and MongoDB
📱 Responsive React interface with reusable components


🧠 Key Decisions


Using React and reusable components made the interface easier to maintain as the project grew.
Separating the frontend and backend allowed for a cleaner API and better-organized application.
MongoDB was a good fit for storing flexible user and habit data.
Adding Google Gemini helped turn habit data into personalized feedback instead of displaying only statistics.
Using Recharts made it easier for users to quickly understand their progress through visualizations.


📚 What This Project Taught Me

Building YouThrive gave me practical experience developing a full stack application from start to finish. I became more confident working with REST APIs, JWT authentication, MongoDB, and connecting a React frontend to an Express backend.
